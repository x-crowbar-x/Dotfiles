## My Dotfiles

In this repository you can find some of the configs I use for my Arch Linux install. 

![This is how it looks like.](/Screenshot.png)

### Dependencies

**Important**
* pacman, enabled AUR (Arch-based Linux distro)
* nm-applet
* blueman-applet
* volumeicon
* picom
* rofi
* dmscripts
* brightnessctl
* alacritty
* pavucontrol (Pulse Audio)
* maim
* xkblayout
* xwallpaper

**Less important**
* firefox
* thunar
* libreoffice
* atom
